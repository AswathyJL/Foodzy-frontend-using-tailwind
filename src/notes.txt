
---------------------------------------------------------------------
                            Tailwind CSS
---------------------------------------------------------------------


                        Spacing
                        *******
Padding - p/px/py/ps/pe/pt/pr/pb/pl - 0/px/0.5/1/1.5/2...
Margin - m/mx/my/ms/me/mt/mb/mr/ml - 0/px/0.5/1/1.5/2...
Space between - space-x/y-0/px/0.5/2/...  (shortcut for adding margin to all-but-the-first-item in a group) use gap for complex wrapong layouts like grid

                        Sizing
                        ******
Width - w-0/px/0.5/1/1.5/2.../auto/(1/2)/(1/3)/(1/4)/(1/5)/full/screen/svw/lvw/dvw/min/max/fit
Height - h-0/px/0.5/1/1.5/2.../auto/(1/2)/(1/3)/(1/4)/(1/5)/full/screen/svh/lvh/dvh/min/max/fit
size - w-0/px/0.5/1/1.5/2.../auto/(1/2)/(1/3)/(1/4)/(1/5)/full/min/max/fit

                        Typography
                        **********
Font family - font-sans / font-serif / font-mono  (customisale)
Font size - text-xs/sm/base/lg/xl/2xl/....../9xl
Font Style - italic / not-italic

                        FlexBox
                        *******
FlexBox - flex flex-row/row-reverse/column/column-reverse 
        - gap-x/y-0/1/0.5/.....
        - justify-content - justify-normal/start/end/center/between/around/evenly/stretch
        - align-content - content-normal/start/end/center/between/around/evenly/stretch/baseline
        - Place-content - place-content-baseline/start/end/center/between/around/evenly/stretch
